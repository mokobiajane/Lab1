\hypertarget{index_autotoc_md11}{}\doxysubsection{\texorpdfstring{4. Parallelism}{4. Parallelism}}\label{index_autotoc_md11}
A custom {\ttfamily parallel\+\_\+for} template distributes work across multiple CPU threads for the following operations\+:


\begin{DoxyItemize}
\item Grayscale conversion
\item Rotation (clockwise \& counterclockwise)
\item Gaussian filtering
\end{DoxyItemize}

Threads are automatically scaled based on {\ttfamily std\+::thread\+::hardware\+\_\+concurrency()}.
